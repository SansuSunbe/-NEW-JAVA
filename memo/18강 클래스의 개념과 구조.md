
- ## 클래스란?
	- 객체 지향 프로그래밍의 기본 단위 
	- 객체를 구현하는 설계도

- ## 클래스의 종류
	- ### 실행용
		- JAVA 언어 기준 main 함수가 있는 클래스
	- ### 라이브러리
		- 목적을 위해서 만들어진 도구
		- 기능 목적을 위해 만들어진 것
		- 정보(함수, 데이터)가 모아진 것

- ## 클래스의 구성
	- #### 필드
		- 속성들의 집합
	- #### 메서드(함수)
		- 기능(코드)들의 집합
	- #### 생성자
		- 클래스 생성 역할

- ## 클래스의 사용
	- 클래스를 사용하기 위해서는 객체(인스턴스)화 해야함
	~~~
	클래스명 변수명 = new 클래스명();
	~~~ 
	- 위의 변수명은 클래스의 주소를 가지고 있으며 "인스턴스"라고 부름
