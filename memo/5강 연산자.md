- ## 증감 연산자 사용법
	- 증감 연산자는 숫자에 바로 사용하는게 아닌 변수의 앞 뒤에 붙여서 사용한다.
	- 증감 연산자가 변수의 앞 또는 뒤 어디에 붙이는냐에 따라 결과가 달라진다.

- ## 비교연산자, 논리연산자 사용 예시
	- > : 크다
	- < : 작다
	- >= : 크거나 같다
	- <= : 작거나 같다
	- == : 같다
	- != : 같지 않다
	- &&(AND) : 비교값이 전부 참이면 결과가 참
	- ||(OR) : 비교값이 둘 중 하나라도 참이면 결과가 참
